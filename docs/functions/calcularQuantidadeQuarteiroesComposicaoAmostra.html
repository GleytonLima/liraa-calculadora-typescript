Function calcularQuantidadeQuarteiroesComposicaoAmostra

  • calcularQuantidadeQuarteiroesComposicaoAmostra(quantidadeImoveisEstrato, quantidadeImoveisProgramados, estimativaImoveisPorQuarteirao): number

  • Recebe como parâmetros a quantidade de imóveis em um determinado estrato, a quantidade de imóveis programados para serem inspecionados, e a estimativa de imóveis por quarteirão. A função retorna a quantidade de quarteirões necessários para compor a amostra de inspeção, considerando a quantidade de imóveis programados e a estimativa de imóveis por quarteirão. Se a quantidade de imóveis no estrato é menor que 8100, devem ser inspecionados 20% dos imóveis Caso contrário, devem ser inspecionados 50% dos imóveis.

    Parameters

    • quantidadeImoveisEstrato: number

      A quantidade de imóveis do estrato.

    • quantidadeImoveisProgramados: number

      A quantidade de imóveis a serem programados para inspeção.

    • estimativaImoveisPorQuarteirao: number

      A estimativa da quantidade de imóveis por quarteirão.

    Returns number

    A quantidade de quarteirões que devem ser sorteados para compor a amostra do estrato.
